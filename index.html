关于量盾

我们的出发点

量盾安全软件是一款面向 Windows 平台的免费安全防护工具，源代码在 GitHub 上公开。项目的初衷很简单：做一款不臃肿、不打扰用户、能切实完成安全防护工作的软件。

当前市面上许多安全软件体积庞大、功能冗余，往往附带大量用户并不需要的附加模块，甚至存在弹窗广告和后台驻留难以关闭的问题。量盾希望以另一种方式来做安全软件——把核心的扫描和查杀做好，其余的交给用户自己决定。

量盾安全中心是主程序，基于 ClamAV 开源反病毒引擎，结合自研病毒库与原生 ClamAV 病毒库，提供实时防护、多种扫描模式、启动项管理、日志记录等功能。量盾HIPS 是一款基于主动防御技术的系统防护工具，通过监控系统行为阻止恶意软件对系统的篡改与破坏。量盾安全浏览器是一款注重安全与隐私的浏览器，为你的上网提供额外一层防护。

项目持续维护中，欢迎通过 GitHub 提交问题反馈和代码贡献。
